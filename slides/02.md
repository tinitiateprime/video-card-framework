# Step 1 — Chunking ✂️
![](https://picsum.photos/seed/rag2/2000/1200)

Split long docs into small chunks:
- 200–500 tokens (common)
- Keep headings + metadata
- Store chunk_id, source, page, url

**Goal:** retrieval finds the *right* piece fast.
